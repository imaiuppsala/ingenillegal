# Engagera dig

## Vi söker alltid personer som vill engagera sig i IMÄI Uppsala.

Du kan engagera dig hos oss genom att:

* Bli kontaktperson till någon som är papperslös. Då hjälper du personen med individuella behov så som vårdkontakter, förmedlar frågor som du själv inte kan svara på vidare till nätverket och förmedlar pengar från nätverket till personen.
* Hjälp till med event! Vi behöver alltid fler som kan hjälpa till att planera och arrangera insamlingsevent, ofta behövs många volontärer som kan baka, affischera eller ställa iordning.
* Arbeta med opinionsbildning för att uppmärksamma papperslösas situation och stärka deras rättigheter.
* Har du plats hemma? En av de vanligaste frågorna vi får är från personer som behöver hjälp med boende. Så om du har ett rum att låna ut för kortare eller längre tid, prata med oss!
* Kom med egna idéer om vad du tycker att vi ska göra!

Om det är något du funderar på, fråga oss gärna!
